# [Flask Material Dashboard PRO](https://appseed.us/admin-dashboards/flask-dashboard-material-pro)

**Commercial product** - Flask **Dashboard** coded on top of **Material Design Pro** crafted by Creative-Tim. **Features**:

- SQLite, PostgreSQL
- SQLAlchemy ORM, Alembic (for db schema migration)
- Modular design with **Blueprints**
- Session-Based authentication (via **flask_login**)
- Forms validation
- Deployment scripts: Docker, Gunicorn
- **24/7 LIVE Support**
